## 1. Initialisation du projet

1. Placez vous dans le répertoire du projet dans un terminal et initialisez git avec la commande :

    ```bash
    git init
    ```
2. Publier le projet sur un dépôt distant **privé** (GitHub, Gitlab, etc.)

    * Si vous utilisez VSCode, vous pouvez le faire directement depuis l'interface graphique
    * Sinon, référez-vous à la documentation de votre IDE et hébergeur git pour savoir comment faire

## 2. Complétion des algorithmes

En prenant exemple sur le code de l'algorithme BFS, complétez le code des autres algorithmes de recherche fournis dans le dossier `app/algorithms`.
