## Barème

Dans ce projet, vous avez une assez grande liberté entre la programmation des tâches de bases, et le développement de programmes approfondis accompagnés d’analyse des résultats : les notes reflèteteront votre niveau d’investissement

Les notes les plus elevées seront attribuées aux projets les plus ambitieux, les plus complets, les mieux documentés et les mieux présentés.

## Plagiat

Le plagiat est le fait d'utiliser le travail de quelqu'un d'autre et de le faire passer pour le votre. 

C'est une **faute grave** qui peut entraîner l’exclusion du Master et l’interdiction de passer des examens pour une durée de 5 ans.

Si vous utilisez une librairie ou le code source d’un autre programmeur dans votre projet, vous devez le préciser dans votre rapport ou dans votre code en commentaires. 

Il est bien sûr admis de discuter entre vous et de s’inspirer de projets existants, mais vous devez réaliser le projet vous-même.

## Exemples de projets

* **Jeux à 2 joueurs** sur des grilles : **Jeu de Go**, **Jeu de dames**, **Echecs**, **Puissance 4**, **Morpion**, etc.
* **Jeux à plusieurs joueurs** sur des grilles : **Jeu de l'oie**, **Jeu de labyrinthe**, **Jeu de plateau**, etc.
* **Autres jeux de plateau** : Trouver un jeu de plateau que vous aimez et qui n'existe pas en version informatique.
* **Jeu en solitaire** sur des grilles : **Sudoku**, **Kakuro**, **Démineur**, **Taquin**, etc.
* **Jeu de cartes** : Belote, Tarot, Poker, etc. _Attention : Les jeux de cartes sont souvent plus complexes à implémenter que les jeux sur des grilles, car les états du jeu sont plus difficiles à représenter._
* **Application de gestion de plannification** : **Agenda**, **Gestion de tâches**, **Gestion de ressources**, etc.
