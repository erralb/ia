## Introduction

Dans ce cours, nous allons découvrir les bases de l'Intelligence Artificielle (IA), et comment elle peut être utilisée pour résoudre des problèmes complexes.

Que vous soyez débutant ou que vous ayez déjà une certaine expérience dans le domaine, ce cours vous aidera à approfondir vos compétences et votre expertise en IA.

<Slide>

Dans ce cours, nous allons aborder :

* Les algorithmes de **recherche** : BFS, DFS, A*, Minimax, Alpha-Beta, etc.
* Les algorithmes de résolution de problèmes (**CSP**)
* La logique de premier ordre (**First Order Logic**)
* Les sytèmes **Multi-Agents**

</Slide>
<Slide>

:::danger[Attention]
Le **Machine Learning** ne sera pas étudié pendant ce cours, l’idée ici est d'aborder les techniques historiques de l’IA
:::

> Cependant, comprendre ces techniques historique permettra une compréhension plus aisée des modèles utilisés pour le machine learning et les modèles d’interaction du langage.


</Slide>
<Slide>

## Compétences visées

* Comprendre les bases de l'Intelligence Artificielle
* Savoir implémenter les algorithmes de recherche
* Savoir comment résoudre des problèmes de contraintes
* Comprendre les systèmes multi-agents
* Développer des compétences en programmation à travers le développement d'un jeu utilisant les techniques étudiées

</Slide>
