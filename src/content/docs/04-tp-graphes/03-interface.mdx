---
title: "Ressources"
---

Voici une liste **non exhaustive** de librairies et de générateurs de graphes pour différents langages de programmation.

## Visualisation

* [Graphviz](https://graphviz.org/)
* [Awesome Dataviz](https://github.com/hal9ai/awesome-dataviz)

## Javascript / Typescript

* [Graphology](https://graphology.github.io/)
* [Graph generator for Graphology](https://www.npmjs.com/package/graphology-generators)
* [Graphviz avec Typescript](https://ts-graphviz.github.io/ts-graphviz/)

## Python

* [NetworkX](https://networkx.org/)
* [Graph-tool](https://graph-tool.skewed.de/)
* [Graph generator for NetworkX](httpsnetworkx.org/documentation/stable/reference/generators.html)

## Java

* [JGraphT](https://jgrapht.org/)
* [Graph generator for JGraphT](https://jgrapht.org/generators/)
* [GraphStream](http://graphstream-project.org/)
* [Graph generator for GraphStream](http://graphstream-project.org/doc/Generators/)
* [Apache Commons Math](https://commons.apache.org/proper/commons-math/)
* [Apache Commons Math - Graph](https://commons.apache.org/proper/commons-math/javadocs/api-3.6.1/org/apache/commons/math3/graph/package-summary.html)

## C++

* [Boost Graph Library](https://www.boost.org/doc/libs/1_76_0/libs/graph/doc/index.html)
* [Graph generator for Boost Graph Library](https://www.boost.org/doc/libs/1_76_0/libs/graph/doc/graph_generators.html)

## C#

* [QuikGraph](https://www.nuget.org/packages/QuikGraph)

## Rust

* [Petgraph](https://docs.rs/petgraph/0.5.1/petgraph/)

## Go

* [gonum](https://pkg.go.dev/gonum.org/v1/gonum/graph)
* [Graph generator for gonum](https://pkg.go.dev/gonum.org/v1/gonum/graph/simple)

## PHP

* [GraphP](https://github.com/graphp/graph)
* [Cytoscape.js](https://js.cytoscape.org/) - une librairie Javascript mais qui peut être utilisée avec PHP

Plus globalement, pour utiliser PHP, il vaut mieux intégrer une librairie en Javascript dans son application PHP pour la visualisation des graphes.

## Ruby

* [Ruby librairies](https://www.ruby-toolbox.com/categories/graphing)


## Swift

* [SwiftGraph](https://github.com/davecom/SwiftGraph)
* [ChartsOrg](https://github.com/ChartsOrg/Charts)

## R

* [igraph](https://igraph.org/r/)
* [Graph generator for igraph](https://igraph.org/r/doc/make_undirected_graph.html)
* [Graph gallery](https://r-graph-gallery.com/)
