<Slide title="Projet">

:::note[Objectif principal]
Le but de ce projet, à faire en binôme ou en trinôme, est d'implémenter un jeu ou une application en utilisant des techniques d'intelligence artificielle d’une part, et d'évaluer comment ces techniques améliorent la performance de votre programme d’autre part.
:::

</Slide>
<Slide>

## Etape 1

Définir une équipe de 2 ou 3 personnes et choisir un jeu ou une application de stratégie utilisant une ou plusieurs techniques d'intelligence artificielle. Vous pouvez travailler seul, mais cela sera plus difficile.

:::danger[Important]
Vous **devez** me faire valider votre choix application, en m'envoyant un email avec des informations sur votre choix, ainsi que les personnes de votre groupe
:::

Vous **devez vous documenter en amont** sur les techniques à utiliser pour votre application (_Logique de Premier Ordre, Théorie des jeux, Agents intelligents, autres algorithmes de recherche heuristique, CSP, etc._) car vous devez commencer à travailler sur le projet avant la fin du cours.

</Slide>
<Slide>

## Etape 2

Implémenter le moteur votre application, sans interface graphique, en utilisant les techniques vues pendant les cours (heuristiques, A\*, minimax, élagage alpha beta, CSP, etc.) et en vous documentant sur d'autres techniques si nécessaire.

   *Vous pouvez utiliser le langage de programmation ou framework que vous souhaitez (Javascript, Typescript, Angular, C++, C#, Java, Python, Rust, Ruby, PHP, GO, etc.).*
   *Attention aux problèmes de synchronisation avec Javascript/Typescript*

</Slide>
<Slide>

## Etape 3

Développer des séries de tests pour mesurer la complexité et les performances de votre application

   * *Par exemple, faire jouer chaque IA 1000 parties de votre jeu). Les statistiques devraient inclure des données de temps d'exécution, d’utilisation de mémoire, de nombres de parties gagnées, etc.*
   * *Est-ce que l'heuristique 1 est meilleure que l'heuristique 2 ?*
   * *Quel est l'effet de la profondeur de l'arbre sur les performances ?*
   * *Quel est le gain d'une version avec élagage alpha-beta par rapport à une version sans ?* 
   * *Quel est l'effet de la taille de la grille sur les performances de votre algorithme ?*
   * *etc.*

</Slide>
<Slide>

## Etape 4

Développer une interface graphique pour que des humains puissent utiliser votre application ou votre jeu et observer vos IA fonctionner.

      *Si votre jeu est un jeu multi-joueurs, vous devez programmer votre jeu pour qu'il puisse jouer contre des humains et contre lui-même avec différents bots.*

      *Ne passez pas trop de temps sur l'apparence de l'interface graphique, quelque chose de simple suffira. Vous pouvez aussi utiliser des librairies existantes pour vous faciliter la tâche.*

</Slide>
<Slide>

## Etape 5

Rédiger un rapport d'au moins 10 pages (en français ou en anglais) décrivant votre application, les techniques d'intelligence artificielle utilisées, les résultats de vos tests, et les conclusions que vous en tirez.

</Slide>
<Slide>

## Tout au long du projet

* Me tenir informer de l'avancement de votre projet toutes les 2 semaines
* Utiliser un outil de gestion de projet (Taiga, Trello, Asana, etc.) et **l'utiliser régulièrement**
* Utiliser Git pour versionner votre code et **publier des commits régulièrement**
* Utiliser si possible un outil de CI/CD pour automatiser vos tests ou préparer des scripts pour automatiser vos tests
* Compléter un **manuel utilisateur** pour votre l'installation et l'utilisation de votre application

</Slide>
